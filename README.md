# Lou's Houdini Shelves
Oi bruv these are my bloody shelves init

## Install
Clone the repo in your `C:/Users/XXXX/Documents/houdiniX.X/toolbar` folder like so :
```bash
cd C:/Users/X_your_user_X/Documents/houdiniX.X/toolbar
git clone https://github.com/lou-delmay/houdini_shelves.git
```

Now Start Houdini and you should see my shelf in the list. It's called `Lou`
Enjoy <3
